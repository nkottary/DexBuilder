# DexBuilder

[![Build Status](https://travis-ci.org/tanmaykm/DexBuilder.svg?branch=master)](https://travis-ci.org/tanmaykm/DexBuilder)

This repository builds binary artifacts for the DexBuilder project. Binary artifacts are automatically uploaded to
[this repository's GitHub releases page](https://github.com/tanmaykm/DexBuilder/releases) whenever a tag is created
on this repository.

This repository was created using [BinaryBuilder.jl](https://github.com/JuliaPackaging/BinaryBuilder.jl)
